Bug Eyes
(c) 1985 Icon Software Ltd.
A ZX Spectrum 48K game

ROM v1.1 - uses ZX0 classic compression
Uses the following original ZX routines:
CHARSET, BEEPER and SCROLL ($0E00) routine
